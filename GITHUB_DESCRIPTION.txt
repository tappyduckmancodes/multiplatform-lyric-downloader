# About section text
A cross-platform CLI tool that downloads time-synced .lrc lyric files for your music library.
Fetches from a 5-source waterfall: Spotify → Deezer → LRCLIB → Musixmatch → YouTube captions.
Output is organized into Artist/Album/Track.lrc folders compatible with MusicBee, Plex, Kodi,
Poweramp, and any LRC-aware media player.

Supports: track, album, playlist, and artist discography downloads from Spotify, Deezer, YouTube,
and YouTube Music URLs. Automatic source fallback with full logging. Cross-platform: Windows
(CMD/PowerShell), macOS, Linux, Git Bash.

Complete rewrite of spotify-lyric-downloader — same output format, no browser automation required.
